# **App Name**: PayrollPal

## Core Features:

- Employee Data Input: Form to input employee data (name, hourly rate, hours worked).
- Payroll Calculation: Automatic calculation of gross pay, taxes (using fixed rates for MVP), and net pay.
- Payroll Display: Clear display of each employee's payroll information, including itemized deductions.
- Login Page: Basic login page.
- Sign up Page: Sign up page
- Main UI: The main UI after login, as described in the attached image.

## Style Guidelines:

- Primary color: Calm blue (#3498db) for trust and reliability.
- Secondary color: Light gray (#f2f2f2) for a clean background.
- Accent: Green (#2ecc71) to indicate success and positive actions (e.g., "Calculate Payroll").
- Clean and readable sans-serif fonts for all text elements.
- Simple, single-page layout for easy navigation.
- Use clear and recognizable icons for key actions (e.g., a calculator for payroll calculation).
